## 필수 요구사항

- Docker
- Docker Compose
- Java 17 이상

## 프로젝트 실행 방법 - Root 디렉토리

```bash
git clone https://github.com/ehdtndla123/Ringle.git

cd Ringle

./gradlew bootJar

docker-compose up --build
```

### 실행 확인

http://localhost:8080/swagger-ui/index.html

### 예시 서버

http://138.2.112.17:8080/swagger-ui/index.html
